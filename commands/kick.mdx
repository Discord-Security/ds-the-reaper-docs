Para usá-lo, digite **/kick**.

<img
  src="https://i.imgur.com/sxfQtBR.png"
  className="mx-auto"
/>

- `Usuário`: Este basta o mencioná-lo no servidor ou usar o ID de usuário.
- `Motivo`: Diga o motivo pela expulsão do usuário.
